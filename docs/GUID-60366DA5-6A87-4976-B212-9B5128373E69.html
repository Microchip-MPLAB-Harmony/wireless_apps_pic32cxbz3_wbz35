4.2 Install Device Family Part Pack

PIC32CXBZ3/WBZ35 Device Support is handled by Device Family Part Pack installation

Installing PIC32CXBZ3/WBZ35 DFP

  1. Open the MPLAB X IDE and Select Tools > Packs
  2. Click on "Instal or Uninstall Packs" → "Install Pack from Local Source"
  3. Browse to EAP1 package \EV19J06A\Tools\Microchip.PIC32CX-BZ3_DFP-1.0.52
  4. Let the DFP installation happen and then restart the MPLABX

Introduction to DFP gives background information on usage of Device Family Part Pack in application development
